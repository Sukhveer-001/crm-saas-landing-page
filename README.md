# CRM SaaS Landing Page

A modern, responsive landing page for a CRM SaaS platform, built with **React** and **Tailwind CSS**. Designed for portfolio showcase and frontend practice, this project demonstrates clean component-based architecture and a polished SaaS UI.

[Live Demo](https://crm-saas-landing-page.netlify.app/)

## Features

- **Responsive Navigation:** Works seamlessly on mobile and desktop  
- **Hero Section:** Engaging headline, subtext, and call-to-action button  
- **Features Section:** Highlights CRM benefits and capabilities  
- **Pricing Section:** Cleanly designed pricing cards for SaaS plans  
- **Call-to-Action:** Encourages users to try or sign up  
- **Modern UI:** Minimal, aesthetic design with thin borders and subtle colors  

---

## Tech Stack

- **React** – Component-based frontend  
- **Tailwind CSS** – Utility-first styling  
- **Vite** – Fast development and build tool  
- **Netlify** – Live deployment
